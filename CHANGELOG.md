# Changelog

## v1.0.0

- Initial release
- Right-click context menu to highlight/unhighlight rows
- Permanent row highlighting (does not disappear when moving cursor)
- Support for multiple cursors (`Alt + click`)
- Configurable highlight color (HEX, RGBA, CSS color names)
- Works with any file type (`.mb`, `.txt`, `.js`, `.py`, logs, configs, etc.)
- Highlight bound to row number (simple and predictable behavior)
- Highlights persist within VS Code session (tab switching, file reopen)
